# 俄罗斯方块 SuperTetris
偶然找到一份大学时候写的垃圾代码-。-

# 安装
```bash
git clone https://github.com/DeaglePC/SuperTetris.git
```

# 运行
*确保你有java环境*
```bash
cd dist
java -jar SuperTetris.jar
```
